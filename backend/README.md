# ReviewGuard AI — Backend

FastAPI-based backend for fake review detection using **SBERT embeddings** (all-mpnet-base-v2, 768-dim) combined with a **Random Forest classifier** and 29 hand-crafted linguistic features.

## Features

- **Single Review Analysis** — classify any text review as Fake or Genuine with probability scores, risk level, detected signals, and a human-readable explanation.
- **Batch Prediction** — classify up to 30 reviews in a single request with aggregated stats.
- **URL-Based Scraping & Analysis** — paste a product URL and the backend scrapes reviews, classifies each one, and returns an overall trustworthiness verdict.
- **Multi-Site Scraper** — supports Amazon (all domains: `.in`, `.com`, `.co.uk`, etc.), Flipkart, Walmart, BestBuy, eBay, and a generic fallback for other sites. Uses `httpx` + `BeautifulSoup` (no browser/Playwright required).
- **29 Hand-Crafted Features** — word count, lexical diversity, exclamation/caps ratio, positive/negative keyword counts, spam phrase detection, credibility claims, VADER sentiment, stopword ratio, dramatic word ratio, and more.
- **Short URL Resolution** — automatically resolves `amzn.in`, `amzn.to`, `a.co`, and similar short links.

## Tech Stack

| Component           | Library / Tool                                                    |
| ------------------- | ----------------------------------------------------------------- |
| Web Framework       | FastAPI + Uvicorn                                                 |
| ML Classifier       | scikit-learn (RandomForestClassifier)                             |
| Sentence Embeddings | sentence-transformers (all-mpnet-base-v2)                         |
| Sentiment Analysis  | vaderSentiment                                                    |
| Web Scraping        | httpx + BeautifulSoup + lxml                                      |
| Feature Scaling     | Custom `SelectiveScaler` (MinMaxScaler on hand-crafted dims only) |

## Prerequisites

- Python 3.9 or higher

## Setup

1. **Install dependencies:**

   ```bash
   pip install -r requirements.txt
   ```

2. **Train the model** (run once before starting the API):

   ```bash
   python train.py
   ```

   This will:
   - Load `Balanced_Final_Dataset.csv`
   - Encode all reviews with SBERT (768-dim)
   - Extract 29 hand-crafted features per review
   - Run `RandomizedSearchCV` for hyperparameter tuning
   - Train a Random Forest pipeline with `SelectiveScaler`
   - Save the model to `rf_model.joblib`

3. **Start the API server:**

   ```bash
   python app.py
   ```

   The server runs at **http://localhost:8000**.

## API Endpoints

| Method | Endpoint         | Description                                  |
| ------ | ---------------- | -------------------------------------------- |
| GET    | `/`              | Service info (name, version, status)         |
| GET    | `/health`        | Health check                                 |
| GET    | `/model-info`    | Model metadata (SBERT name, dims, threshold) |
| POST   | `/predict`       | Classify a single review                     |
| POST   | `/batch-predict` | Classify up to 30 reviews at once            |
| POST   | `/analyze-url`   | Scrape & classify reviews from a product URL |

### `POST /predict`

**Request:**

```json
{
  "review": "This product is absolutely amazing, best purchase ever!!!"
}
```

**Response:**

```json
{
  "prediction": "Fake",
  "confidence": 0.87,
  "genuine_probability": 0.13,
  "fake_probability": 0.87,
  "risk_level": "High",
  "features": { "word_count": 8, "exclamation_marks": 3, "...": "..." },
  "signals": ["3 exclamation marks — high emotional intensity"],
  "description": "This review is classified as FAKE with 87.0% probability..."
}
```

## Training Details

- **Embeddings:** all-mpnet-base-v2 (768 dimensions, unit-normalized)
- **Hand-crafted features:** 29 linguistic / stylometric signals
- **Total feature vector:** 797 dimensions (768 + 29)
- **Classifier:** Random Forest (500 trees, `class_weight={0:1, 1:2}` to penalize fake misses)
- **Tuning:** `RandomizedSearchCV` with 12 iterations, 3-fold stratified CV, optimized on F1 score
- **Evaluation:** 5-fold cross-validation, test accuracy target ≥ 90%
- **Fake threshold:** 0.40 (reviews with ≥ 40% fake probability are classified as Fake)
